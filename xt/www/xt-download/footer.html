Installing a Binary Distribution of XT

  • Download the binary distribution and save it to disk.
  • Install by unpacking the distributioin (XT will be installed in /opt) 
        # cd /
    # gunzip xt-<version>.<os>.<arch>.tar.gz
    # tar xvf xt-<version>.<os>.<arch>.tar

Building and Installing the Source Distribution of XT

  • Download the source distribution and save it to disk.
  • Unpack the distribution: 
         # gunzip xt-<version>.src.tar.gz
     # tar xvf xt-<version>.src.tar
  • cd to the source directory: 
         # cd xt-<version>
  • Read the installation instructions contained in the top-level README file.
