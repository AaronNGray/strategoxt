Installing a Binary Distribution of XT

  • Installing a tar package

    1. Download the binary distribution and save it to disk.
    2. Become root
    3. Install by unpacking the distribution (XT will be installed in /opt) 
          # su
    # cd /
    # gunzip xt-<version>.<os>.<arch>.tar.gz
    # tar xvf xt-<version>.<os>.<arch>.tar

  • Installing a Linux rpm package

    1. Download the binary distribution and save it to disk.
    2. Become root
    3. Install by unpacking the distribution and installing it using the rpm package manager (XT will be installed in /opt) 
          # su
    # rpm -i xt-<version>.<os>.<arch>.rpm

  • Installing a Sun pkg package

    1. Download the binary distribution and save it to disk.
    2. Become root
    3. Install by unpacking the distribution and installing it using ``pkgadd'' (XT will be installed in /opt) 
          # su
    # gunzip xt-<version>.<os>.<arch>.pkg.tar.gz
    # tar xvf xt-<version>.<os>.<arch>.pkg.tar
    # pkgadd -d .

Building and Installing the Source Distribution of XT

  1. Download the source distribution and save it to disk.
  2. Unpack the distribution: 
         # gunzip xt-<version>.src.tar.gz
     # tar xvf xt-<version>.src.tar
  3. cd to the source directory: 
         # cd xt-<version>
  4. Read the installation instructions contained in the top-level README file.
